Overview

A blockchain-based loyalty platform for repair businesses. RepairCoin connects service providers with customers through a token-based reward system, enabling cross-business loyalty redemption on the Base blockchain.

Key Features

  • Marketplace discovery for nearby repair services
  • Cross-business token redemption system
  • Stripe payment integration
